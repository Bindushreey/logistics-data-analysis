# Logistics Data Analysis (Python)

This project performs exploratory data analysis on logistics and supply chain datasets to understand data structure, quality, and operational patterns.

## Files
- eda.py : Python script used for data cleaning, preprocessing, and analysis

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Key Analysis Performed
- Dataset structure exploration
- Duplicate removal
- Null value checks
- Basic statistical insights

This project demonstrates hands-on experience working with operational datasets using Python.
